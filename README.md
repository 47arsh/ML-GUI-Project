# Machine Learning GUI Studio

A Streamlit-based graphical interface for experimenting with machine learning algorithms using built-in datasets, uploaded CSV files, manually entered data points, visualizations, model comparison, and export tools.

## Project Structure

```text
ml-gui-project/
|-- app.py
|-- requirements.txt
|-- README.md
`-- DOCUMENTATION.md
```


```powershell
cd F:\ml-gui-project
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Run

```powershell
streamlit run app.py
```

The app opens in the browser at the local Streamlit URL printed in the terminal.
